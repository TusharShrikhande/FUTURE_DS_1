# FUTURE_DS_1
📊 Business Sales Performance Analytics Dashboard

This project presents an interactive Sales Performance Analytics Dashboard built using Power BI, designed to uncover key business insights from retail transaction data. The dashboard enables users to analyze sales trends, product performance, and country-wise contributions in a visually intuitive manner.

🚀 Project Overview

The goal of this dashboard is to transform raw sales data into meaningful insights that support data-driven decision-making. It focuses on identifying top-performing products, understanding geographic sales distribution, and tracking overall revenue trends.

📌 Key Features
📈 Sales Trend Analysis
Visualizes the Sum of Unit Price across different years and countries.
Helps identify growth patterns and major contributors (e.g., United Kingdom, Germany, France).
📦 Product Performance Insights
Displays Sum of Quantity sold by Products.
Highlights best-selling items like POSTAGE, WHITE HANGING HEART, etc.
🌍 Country-Level Analysis
Compares sales contributions across multiple countries.
Useful for identifying high-performing markets.
📊 KPI Metrics
Total sales value (Sum of Unit Price)
Individual product-level pricing insights
🔍 Interactive Filtering
Drill-through and filtering capabilities (e.g., Quantity filter applied)
Enables focused analysis on specific conditions.
